 # ShopEasy - E-Commerce Web Application

A basic online store with product management and order tracking.

## Features
- Product catalog with Add to Cart functionality
- Cart management with quantity and total
- Order placement and order tracking
- Built with Node.js backend API

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Storage: localStorage
